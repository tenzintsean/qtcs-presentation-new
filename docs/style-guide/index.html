Accessibility Guidelines

WCAG 2.1 AA Color Contrast

All text colors meet WCAG 2.1 Level AA contrast requirements (minimum 4.5:1 for normal text, 3:1 for large text).

Usage Color Contrast Ratio Status
Headings #0047AB ~8:1 ✓ AAA
Body Text #1a5490 ~6:1 ✓ AA
Captions/Muted #3a6fa8 ~4.5:1 ✓ AA
Light on Dark #FFFFFF >10:1 ✓ AAA

Do: Use text-primary for headings

Body text uses text-secondary for comfortable reading.

Muted text for captions and metadata.

Do: Use white on dark backgrounds

For footer and dark overlays, white text provides excellent contrast.

Muted light text for secondary info.
